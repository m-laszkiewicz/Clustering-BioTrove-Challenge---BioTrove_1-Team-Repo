# Overview
This repo covers the "BioTrove_1" team approach to the 2025 ML+X Machine Learning Marathon Clustering-BioTrove challenge. The BioTrove_1 team included 3 members: Chenchen D., Zhixing X., and Michal L. The Clustering-BioTrove challenge involves a subset of the original BioTrove biodiversity dataset which contains over 160 million images of living organisms with taxonomic information included. The subset simply contained 50k images and only taxonomic family labels. The challenge is to use unsupervised learning (no genus or species labels provided) to produce genus- and species-level clusters. The included family labels can be used to improve clustering performance into the respective taxonomic level-representative groups. The README file will include information on the original BioTrove dataset, the Clustering-BioTrove challenge and dataset, and the approaches that were employed by our team during the challenge period (Fall 2025).
