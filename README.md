# Base_Model_V2
This is a jupyter notebook file showing the code for the second Biotrove_1 team model for the 'Clustering Biotrove' machine learning challenge. This model extracts features at two separate layers of a pretrained ResNet 50 model with ImageNet weights, and then uses each set of embeddings to cluster the data into genus and species-level groupings using K-Means MiniBatch.
